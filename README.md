# calcul de moyenne et de covariance sur un ensemble d images regrouper en vecteur (64*64,)
Les images regrouper proviennent d'un dataset composer de **400** images de face. chaque image a ete decomposer en **vecteur** pixel par pixel.Les images ont ete grisée pour eviter d augmenter la dimensionnalité des vecteurs a cause du RGB.
Le calcul des moyennes a permis de deceler la **face moyenne** parmis ce lot d'image tandis que la covariance montre la dispersion des pixels d origine par rapport au pixel moyen.
